# IDEA-301
我想制作一个Hackathon比赛平台项目，角色有主办方、参赛者和赞助商这三方，我总共想分成若干期完成这个项目
主办方和赞助商共用一个平台Hackathon Admin Platform，参赛者使用另一个平台Hackathon Arena Platform
## 第三期
第三期完成如下内容
### 1.赞助商入驻流程
#### Admin平台
##### 赞助商
+ Admin平台会提供给赞助商一个申请的页面，赞助商无需登录，可以提交赞助申请
+ 赞助商有两种形式，一个是提供长期赞助，所有活动都会有赞助商的Logo；另一类是赞助商指定赞助某一个活动，赞助商在提交申请时可选择，选择第二类时需要勾选处于发布环节的活动
+ 赞助商在申请时需要填写手机号和赞助商Logo
+ 赞助商可以填写手机号查看申请结果
##### 主办方
+ Admin用户可以审核赞助商的申请，申请通过后会给赞助商自动创建赞助商账号
#### Arena平台
+ Arena平台会在全局展示长期赞助商
+ Arena平台会在指定活动展示指定赞助商
### 2. 活动流程优化
#### Admin平台
##### 创建活动页面
+ 奖项除了包含奖金，还包含奖品，主办方也可以设置，奖品默认填写活动NFT
+ 活动开始时间的时分秒为00:00:00，结束时间的时分秒为23:59:59
+ 在活动创建时根据活动开始时间和结束时间，自动分配其他各阶段开始时间和结束时间
##### 活动详情看板
+ 描述字段展示的样式与其他字段保持一致
+ 统计信息，点击报名人数、签到人数、队伍数量和数量，可以看到详情
##### 活动管理页面
+ 活动管理页面增加活动名称模糊搜索框，动态查询
+ 去除排序搜索框
##### 活动发布
