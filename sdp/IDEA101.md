# IDEA-101
我想制作一个Hackathon比赛平台项目，角色有主办方、参赛者和赞助商这三方，我总共想分成若干期完成这个项目
主办方和赞助商共用一个平台Hackathon Admin Platform，参赛者使用另一个平台Hackathon Arena Platform
## 第一期
第一期完成如下内容
### 主办方
+ 主办方可以通过账号密码形式登录Hackathon Admin Platform
+ 主办方有一个Admin账号可以添加主办方人员和赞助商人员
+ 主办方可以在Hackathon Admin Platform增删改查活动
+ 活动分为预备、发布、报名、签到、组队、提交、投票、公布结果这几个阶段
+ 其中报名、签到、组队、提交、投票这几个阶段需要开始和结束时间，在这个阶段的时间范围内只能做相应的事
### 参赛者
+ 参赛者通过关联Metamask钱包登录Hackathon Arena Platform
+ 参赛者可以在Hackathon Arena Platform看见主办方创建并发布的活动，并报名参加，此操作无需主办方审核
+ 参赛者报名审批通过后，在活动签到阶段，可以进行签到，此操作无需主办方审核
+ 参赛者完成签到后，在组队阶段，可以进行组队，此操作无需主办方审核
+ 参赛者组队完成后，在提交阶段，可以提交作品，作品仅能由队长提交，此操作无需主办方审核
+ 参赛者提交作品后，在投票阶段，可以对所有参赛队伍提交的作品进行投票，此操作无需主办方审核
+ 参赛者完成投票后，在公布结果阶段，参赛者可以看到结果
### 赞助商
+ 主办方可以通过账号密码形式登录Hackathon Admin Platform
