# IDEA-201
我想制作一个Hackathon比赛平台项目，角色有主办方、参赛者和赞助商这三方，我总共想分成若干期完成这个项目
主办方和赞助商共用一个平台Hackathon Admin Platform，参赛者使用另一个平台Hackathon Arena Platform
## 第二期
第二期内容主要包含对第一期内容的一些优化和修改
## PART1
## Admin平台
### 登录页面
+ 支持两种登录方式，一种是现有的用户名、密码模式；另一种是可以直接连web3钱包登录
+ 将现在的邮箱、密码改为手机号、密码，手机号不能重复，去除邮箱字段
+ 一个角色只能有一个手机号，但可能有多个web3钱包地址
### 个人中心
+ 用户除了角色字段不能修改，其他都可以修改
### 首页
+ Admin角色可以看到仪表盘、人员管理两个页面
+ 主办方角色可以看到仪表盘、活动管理两个页面
+ 赞助商角色可以看到仪表盘一个页面
### 人员管理列表页
+ 已经删除的人员可以看到，用不同的状态显示
+ 添加复原操作，可以重新添加回已删除的人
+ 优化操作栏的前端显示，浏览器非全屏时也能都显示
### 创建活动页面
+ 活动时间精确到日
+ 当地点类型为线下或混合时，需要选择城市，提供默认的城市列表
+ 当地点类型为线下或混合时，需要填写具体地址，具体地址提供地图组件
+ 活动创建页需要设置奖项，奖项包含奖项名称，奖项金额，默认为一个一等奖，两个二等奖，三个三等奖，金额为1000USD、500USD和200USD
+ 活动描述字段不需要为富文本框
